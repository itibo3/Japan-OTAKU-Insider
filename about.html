About Japan OTAKU Insider

Japan OTAKU Insider is a free, English-language database that tracks Japanese otaku culture in near real-time. Built for overseas fans who want to stay up to date without reading Japanese.

What We Cover

Articles are sorted into 7 categories:

  • Collab Cafes: Anime/game-themed pop-up cafes and restaurants across Japan.
  • Figures: Nendoroids, scale figures, prize figures, model kits, and merchandise announcements.
  • Events: AnimeJapan, Comiket, exhibitions, and other otaku events.
  • Anime News: New series, adaptations, broadcast schedules, and industry updates.
  • VTuber: Virtual YouTuber news, live events, and collaborations.
  • Game News: Japanese game releases, updates, esports, and gaming culture.
  • OTAKU NEWS: Akihabara reports, cross-category otaku culture news, and more.

How It Works

Every day, a fully automated pipeline collects, translates, and publishes articles:

  • RSS collection: 21 Japanese RSS feeds are fetched automatically (Natalie, 4Gamer, Amiami, Collabo Cafe, Dengeki Hobby, MoguLive VR, and more).
  • AI-powered search: Perplexity AI searches 6 topic areas to catch stories that RSS misses.
  • Translation: DeepL API translates titles and descriptions into English.
  • Thumbnails: Article images are extracted from each source page automatically.
  • Publishing: Everything is committed to GitHub and deployed to this site via GitHub Pages — zero server cost.

27 sources are registered in total. 6 sources without RSS feeds are covered by the AI search layer.

Tech Stack

  • Static site on GitHub Pages (HTML / CSS / Vanilla JS)
  • Data stored in a single JSON file (entries.json)
  • Daily updates via GitHub Actions (RSS + Perplexity + DeepL + auto-commit)
  • New articles posted to X (Twitter) automatically
