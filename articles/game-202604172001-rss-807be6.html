Details (EN)

The legendary home game console "NEOGEO" has been revived as "NEOGEOAES+", and reservations are now being accepted for release on November 12, 2026. It is a collaboration with SNK, the developer of NEOGEO, and features specifications that focus on reproducing the actual machine. You can play the game with high fidelity that is not an emulation, and it is compatible with original cartridges and peripherals.
